如何将github page与域名绑定

发表于 2017-03-19 | | 阅读次数

若你不舍得花几十块钱一月买服务器，但是舍得花几十块钱一年买域名的话，可以将github page与自己的域名进行绑定，这样自己的博客就可以取一个高大上的网址了。

上一篇文章说了如何创建自己的博客，详情可见三步建立自己的博客，这次我来介绍如何把github的博客跟自己申请的域名进行绑定，让你拥有自己个性名称的个人网站。

申请域名

这一步比较简单，建议用godday的域名，省去备案的麻烦。至于如何申请可百度或谷歌。

创建网址配置文件

方法一：在站点source目录下面，新建一个名为CNAME的文件，里面只写入你要绑定的域名，比如mhwang7758.com（注意，这里为了方便很多人开始的时是创建文本文件.txt，输入完成后一定要记得把后缀名.txt去掉，不然访问不了的）
方法二：在创建Repository的根目录下面，新建一个名为CNAME的文本文件，里面写入你要绑定的域名，比如mhwang7758.com。

DNS配置

申请域名后需要把域名和github的ip地址映射起来，这样访问域名的时候就能跳转到github页面。首先是获取github的ip地址，可以用ping命令获取。打开cmd命令，输入ping 你的github名+github.io。如图1：

图 1. 获取github ip地址

获取ip后，建议使用DNSpod来进行域名的绑定，注册后，添加github的域名和Ip地址，如图2：

图 2. 往DNSpod添加域名和地址

图中记录类型为A的记录值改为你自己上一步获取到的ip地址，主机记录为www开头的填写自己github页面的域名，如mhwang7758.github.io.，io后面的.应该不加也可以，暂时没试过,有兴趣的可以试下。

修改Godaddy的DNS地址

以上步骤准备好后，打开goddady官网,在左上角选择简体中文的语言，登陆成功后点击右上角的用户名，在弹出的窗口中选择管理域名,如图3:

图 3. 选择管理域名

在弹出的窗口中找到自己申请的域名，然后点击域名旁边的小三角按钮，在弹出的对话框中选择设置域名服务器选项，如图4：

图 4. 设置域名服务器

之后在弹出的窗口中选择第二项定制,然后点击添加域名服务器，输入如图4的字段即可:

图 5. 选择域名服务器

如果一切顺利的话，恭喜你，赶紧输入自己的域名看看效果吧！
